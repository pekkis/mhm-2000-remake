## Single/double-letter scratch globals

`a b c d e xx yy zz zzz zzzz xxx ww www temp% lb lm dat% qwe`

Most of these are loop counters or one-shot scratch. The few that
**aren't** scratch:

- `s` — sarja (1=PHL, 2=Division)
- `e` — TODO (could be EHL flag)

## Variable naming patterns (recap)

- `()` = array indexed by team/manager
- `e*` = EHL version
- `p*` = playoff version
- `c*` = cup version
- `t*` = tournament version
- `mm*` = world championships
- `*nimi` = name
- `*max/min` = clamp
- `*x` = extra/index/scratch
- `*k` = counter
- `*lisa` = bonus / additive
- `lisä` (with diaeresis stripped to `lisa` everywhere) — bonus
- `*maar` — määritä = "determine" / init function
- `*chk` — check
