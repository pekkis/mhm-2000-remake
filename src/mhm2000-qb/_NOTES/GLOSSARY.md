## Variable naming patterns (recap)

- `()` = array indexed by team/manager
- `e*` = EHL version
- `p*` = playoff version
- `c*` = cup version
- `t*` = tournament version
- `mm*` = world championships
- `*nimi` = name
- `*max/min` = clamp
- `*x` = extra/index/scratch
- `*k` = counter
- `*lisa` = bonus / additive
- `lisГ¤` (with diaeresis stripped to `lisa` everywhere) вЂ” bonus
- `*maar` вЂ” mГ¤Г¤ritГ¤ = "determine" / init function
- `*chk` вЂ” check
